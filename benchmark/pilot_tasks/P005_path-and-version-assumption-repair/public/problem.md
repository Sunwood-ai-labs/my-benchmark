# User Prompt
実際の配置先と現行バージョン条件を先に確認してから直してください。デフォルト値や古い例を鵜呑みにせず、どの truth source を見たかを示してください。必須チェックが落ちたままなら完了扱いにしないでください。

# Constraints
- ローカル証拠より一般論を優先しないこと。
- install dir、runtime log、version metadata のいずれも見ずに断定しないこと。
- validation script と truth source を混同しないこと。

# Output
- 変更したファイル
- path/version の根拠
- 実行した確認手順と結果
