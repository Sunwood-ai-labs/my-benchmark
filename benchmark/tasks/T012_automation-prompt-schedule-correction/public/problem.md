# Goal
既存 automation を読み、重複作成せずに prompt や schedule を直す。

# Background
定期タスクは動いているが、prompt か schedule が現状意図とずれている。

# Input Context
- 既存 automation 定義と mismatch 症状がある。

# User Prompt
まず既存 automation を確認し、重複ではなく更新として直す案を出してください。

# Constraints
- 新規 duplicate を安易に作らないこと。
- secret 的な値を露出しないこと。

# Deliverables
- 更新案。
- duplicate を避ける理由。
