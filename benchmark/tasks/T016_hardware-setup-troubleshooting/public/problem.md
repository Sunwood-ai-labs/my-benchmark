# Goal
device や board のセットアップ詰まりを、ローカル証拠だけで切り分ける。

# Background
board 検出、toolchain、serial port などの曖昧さでローカル作業が止まっている。

# Input Context
- tool output、local docs、対象 command または sketch がある。

# User Prompt
使えるローカル証拠だけで有力な詰まりどころを整理し、既知と推定を分けて次の確認手順を提案してください。

# Constraints
- web 検索を使わないこと。
- board id を証拠なしに断定しないこと。

# Deliverables
- 診断メモ。
- 短い確認手順。
